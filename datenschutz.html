Grundsatz

Die Website richtet sich derzeit primär an Personen und Organisationen in der Schweiz. Personendaten werden nur bearbeitet, soweit dies für den Betrieb der Website, die Beantwortung von Anfragen oder die technische Sicherheit erforderlich ist.

Diese Daten dienen dem sicheren und stabilen Betrieb der Website, der Fehleranalyse und dem Schutz vor Missbrauch. Eine Auswertung zu Marketingzwecken findet über diese Website nicht statt.

Die Website wird aktuell über GitHub beziehungsweise GitHub Pages bereitgestellt, soweit sie öffentlich aus dem Git-Repository veröffentlicht wird. Dabei können technische Zugriffsdaten durch GitHub verarbeitet werden. Bei einem späteren Wechsel zu einem anderen Hosting-Anbieter wird diese Datenschutzerklärung entsprechend angepasst.

Diese Website verwendet derzeit keine eigenen Cookies, keine Webanalyse-Dienste und keine Marketing-Tracker. Es gibt kein Kontaktformular und keine Newsletter-Anmeldung.

Kontakt per E-Mail oder Telefon

Wenn Sie per E-Mail oder Telefon Kontakt aufnehmen, werden die von Ihnen mitgeteilten Angaben bearbeitet, um Ihre Anfrage zu beantworten und die Kommunikation nachvollziehbar zu führen. Diese Daten werden nur so lange aufbewahrt, wie es für die Bearbeitung der Anfrage, eine spätere Anschlusskommunikation oder gesetzliche Aufbewahrungspflichten notwendig ist.
